SizeRange

data class SizeRange(val minSize: TextUnit, val maxSize: TextUnit, val stepGranularity: TextUnit) : TextScale

Provide a fine-grained configuration for text auto sizing.

Parameters

minSize

the minimum text size available for auto-size. Must be greater than 0 in text units.

maxSize

the maximum text size available for auto-size. Must be greater than 0 in text units.

stepGranularity

the auto-size step granularity in text units. It is used in conjunction with the minimum and maximum text size in order to build the set of text sizes the system uses to choose from when auto-sizing.

Throws

IllegalArgumentException

if any of the auto text-sizing configuration params are invalid.
