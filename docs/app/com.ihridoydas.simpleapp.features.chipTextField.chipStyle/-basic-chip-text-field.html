A text field can display chips, press enter to create a new chip.

Parameters

state

Use rememberChipTextFieldState to create new state.

onSubmit

Called after pressing enter key, used to create new chips.

modifier

Modifier for chip text field.

enabled

Enabled state, if false, user will not able to edit and select.

readOnly

If true, edit will be disabled, but user can still select text.

readOnlyChips

If true, chips are no more editable, but the text field can still be edited if readOnly is not true.

isError

Error state, it is used to change cursor color.

keyboardOptions

See BasicTextField for the details.

textStyle

Text style, also apply to text in chips.

chipStyle

Chip style, include shape, text color, background color, etc. See ChipStyle.

chipVerticalSpacing

Vertical spacing between chips.

chipHorizontalSpacing

Horizontal spacing between chips.

chipLeadingIcon

Leading chip icon, nothing will be displayed by default.

chipTrailingIcon

Trailing chip icon, by default, a CloseButton will be displayed.

onChipClick

Chip click action.

onChipLongClick

Chip long click action.

colors

Text colors. TextFieldDefaults.textFieldColors is default colors.

decorationBox

The decoration box to wrap around text field.
